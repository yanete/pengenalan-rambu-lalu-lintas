Tentang Aplikasi Pengenalan Rambu Lalu Lintas

	Aplikasi pengenalan rambu lalu lintas ini dibuat untuk mengenalkan rambu-rambu lalu lintas kepada semua pengguna jalan. Aplikasi ini dibuat berbasis android sehingga praktis untuk digunakan oleh pengguna jalan karena  mudah di akses dimana saja dan kapan saja. Di samping itu para pengguna jalan akan terbantu dalam mengartikan rambu-rambu lalu lintas saat berada di perjalanan.
	Cara kerja aplikasi ini adalah menunjukkan gambar beserta arti dari gambar tersebut. Aplikasi ini dibuat sedemikian rupa sehingga pemakai akan mudah menggunakan dan menghafalkan rambu-rambu lalu lintas. Selain itu tampilan dari aplikasi ini akan dibuat semenarik mungkin dan user friendly sesuai dengan kebutuhan para pengguna aplikasi.
